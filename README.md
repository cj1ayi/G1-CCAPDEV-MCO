# AnimoForums  

## About
**AnimoForums** is a Reddit-style web forum designed specifically for DLSU students.

Instead of relying on fragmented Discord servers, Messenger group chats, or Facebook groups, the platform centralizes discussions into structured, interest-based Spaces. Students can create posts, engage in threaded conversations, and interact through a voting system вҖ” all within a unified and searchable environment.

It functions as a digital tambayan built for Lasallians:
organized, searchable, and community-driven.


## Tech Stack
`React` вҖў `Vite` вҖў `JavaScript` вҖў `CSS` вҖў `ESLint`
